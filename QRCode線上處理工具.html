功能使用說明

QR Code 掃描

攝影機掃描

點擊「開始掃描」按鈕，允許瀏覽器使用攝影機，將QR Code對準攝影機即可自動識別

圖片上傳掃描

點擊上傳區域選擇圖片檔案，或直接拖拽圖片到上傳區域，系統會自動掃描QR Code

精準掃描區域

如果圖片掃描失敗，可點擊「調整掃描區域」，手動框選QR Code位置進行精準掃描

QR Code 生成

文字QR Code

在輸入框中輸入文字、網址或任何內容，選擇尺寸和顏色後點擊生成

添加Logo標誌

點擊「選擇Logo」上傳圖片，Logo會自動置中並添加白色背景以確保掃描效果

Wi-Fi QR Code

輸入Wi-Fi名稱、密碼和加密類型，生成後掃描即可自動連接網路

使用提示

  • • 支援明暗主題切換，點擊右上角切換按鈕
  • • 掃描結果可複製到剪貼簿或直接開啟連結
  • • 生成的QR Code可下載為PNG圖片或複製到剪貼簿
  • • 所有操作統計會自動儲存在本地
  • • 建議使用高對比度的顏色組合以提高掃描成功率
